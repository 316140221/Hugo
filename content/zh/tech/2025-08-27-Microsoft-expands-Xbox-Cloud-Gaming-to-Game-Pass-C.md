Microsoft is expanding its Xbox Cloud Gaming technology to Xbox Game Pass Core or Standard subscribers, dropping the requirement for the highest tier Xbox Game Pass Ultimate for access to cloud games. As part of an Xbox Insider test, Microsoft is also providing access to some PC games for Game Pass Core and Standard subscribers. [&#8230;]
